# Inspiration
This project was inspired by the challenges high school students face when navigating college admissions. We wanted to simplify the process by creating an interactive platform that provides guidance, resources, and real-time chatbot assistance.

We also incorporated environmental awareness by introducing carbon emission estimates based on travel methods to university.

# What it does
This platform includes multiple web applications designed for student support:

- **How-to-Marymount Experience:** Interactive guided tour of the university website  
- **Admission Form:** Online application form with validation and structured input fields  
- **Chatbot Integration:** AI-powered chatbot to answer student questions in real time

# How it was built
- Frontend built using **HTML, CSS, and JavaScript**  
- Chatbot developed using **Azure OpenAI / Copilot Studio**  
- Integrated multiple pages into one cohesive student experience  
- Presentation video created using After Effects  

Project structure includes:
- `website/` → Main web application files  
- `video/` → Project demo and presentation  
- `presentation/` → Slides and supporting materials
