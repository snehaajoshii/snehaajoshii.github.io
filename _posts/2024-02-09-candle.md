## Project Overview
The Coffee Recommender is a fun and interactive web application designed to help users discover their perfect coffee based on their preferences for strength, milk, and sweetness. The app uses a simple quiz interface to guide users and provides a coffee suggestion based on their choices.

## Features
- **Interactive Quiz:** Users answer three questions to determine their coffee preference.  
- **Coffee Recommendations:** Suggests popular coffee types like Espresso, Latte, Cappuccino, Americano, Macchiato, and Flat White.  
- **Responsive Design:** Works seamlessly across desktop and mobile devices.  
- **Separate Files:** HTML, CSS, and JavaScript are in separate files for modularity and maintainability.  

## Technology Stack
- **HTML5** – Structure and markup for the quiz.  
- **CSS3** – Styling, layout, and responsive design.  
- **JavaScript** – Logic for processing quiz responses and determining coffee recommendations.  

## How It Works
1. Users select their preferences for coffee strength, milk, and sweetness.  
2. Clicking **Submit** triggers the JavaScript logic to determine the recommended coffee.  
3. The recommendation is displayed on the screen, and the quiz can be reset using the **Retry** button.  
4. JavaScript handles input validation to ensure all questions are answered before showing results.  

## File Structure
- `index.html` – Main HTML file with the quiz structure.  
- `style.css` – CSS file for styling and responsive design.  
- `script.js` – JavaScript file for quiz logic and coffee recommendations.  
- `README.md` – Project documentation and instructions.  

## GitHub Pages Deployment
To host this project online using GitHub Pages:  
1. Go to your repository on GitHub.  
2. Click **Settings → Pages** in the sidebar.  
3. Under **Build and Deployment**, select the branch (usually `main`) and folder (`/ (root)`), then click **Save**.  
4. GitHub will generate a live URL for your coffee quiz.  
5. Share the URL so others can take the quiz online.
